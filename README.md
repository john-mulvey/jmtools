# jmtools

Personal utility functions for R data analysis workflows.

## Installation

```r
# install.packages("devtools")
devtools::install_github("john-mulvey/jmtools")
```

## Features

- Custom ggplot2 themes and plotting utilities
- PCA and exploratory data analysis helpers
- Code generation for common analysis workflows (limma, GSEA, ORA)
- Enrichment analysis visualisation functions
